# DIY



## 关系户专属
本校导师直推本校导师直推关键在于通过自己学校的导师的关系来找科研机会。可以看看校内自己认识的熟悉的老师有没有海外合作者？发paper的时候是不是还和其他人合作了？或者看看校内老师的简历中是否有提及去某校访问交流、去某实验室当访问学者、去某校参与XX学术会议。然后和自己的校内老师谈谈，看看能不能让老师帮忙把自己推荐过去。校内导师可能直接帮你问了，或者在同意的情况下让你和对面直接联系。 这条途径的优点在于：靠谱，不会遇到坑的几乎百分百回复率和极高的成功率暑期科研的过程中老板对你的印象会更深刻，毕竟是熟人推荐过来的做得好的话不仅给外国导师留下好印象，还会增加校内导师的好感缺点：在很大程度上取决于校内资源，机会千载难逢需要和校内老师熟悉并且老师很认可你的科研能力

## 小机灵专属

抓与海外院校教授交流的机会这里指的是通过各种非官方非人情的途径抓住和海外教授直接交流的机会，举个例子，XX教授正好来你所在的大学/你所在的城市参加学术会议/参观/旅游，你是不是可以在散会后直接和教授表达你想去他的lab做暑期科研的想法呢？这个途径的成功率取决于①你会不会说话。你可以先从教授交流分享的研究内容开始聊起，展现你对这一领域的认识和思考，然后合理的表达出想去暑期交流的愿望，这是比较合理的。但如果你直接蹿上去要求人家给你暑期科研的机会，八成就没戏了。②对方的实验室需不需要RA。这点看你的运气了

## 海投邮件陶瓷
建议时间规划

1）前一年(2022)10-12月：提前了解科研方向，准备相关申请材料

2）当年1-2月：搜索匹配实验室和教授，发送套瓷邮件

3）当年3-4月：拿到邀请函，办理相关签证

4）当年3-5月：拿到签证，处理机票住宿等

申请材料清单

1）个人简历

2）英语考试，参考UCLA标准，TOFEL 90+，IELTS 7.0+

3）英文成绩单

4）护照

5）个人陈述或动机信（Proposal or Motivation letter）

申请步骤

1）先准备好语言成绩、套磁信和个人简历

2）从各院校官网搜索目标教授的信息和研究方向，向教授发送套瓷信和个人简历

3）收到套瓷回复，研究教授当前研究方向和阅读相关文献，继续学术沟通争取面试和暑研机会

4）拿到暑研机会后尽早(一般至少需要60天)办理J1签证，如无法办理J1签证再考虑B1/B2签证

陶瓷要有针对性的，并不是群发，老师一般有shortlist, 会考虑学生的动机。对于教授来说，给学生暑研机会是一种投资，关心这份投资会不会有回报。 如果这个学生来的目的不是为了以后读PhD，宁愿把这份钱给一个想读PhD，但是能力没有那么强的学生。

所以其实大家套磁的时候， 如果你是想通过暑研争取这个组的PhD， 一定要说出来。 大家可能觉得这样会不会太功利？其实不会的，老师是希望学生能够很简单直白的说出自己的动机，彼此都节省时间。

需要想好自己感兴趣的方向是什么，然后再去这个领域找lab。自己的研究兴趣和lab的研究主题的匹配是非常关键的，这也会对之后的申请有所帮助。如何找到自己的喜欢的实验室并且了解这个实验室主要是做什么的呢？首先还是用我们之前的搜索方式：直接在谷歌（或百度）中搜索【学校名+部门名】，进入部门的网站首页，很多部门都会放一个labs列表，帮助学生快速了解这个部门下都有哪些实验室。建议大家先想好自己以后graduate study阶段想做什么方向，然后照着这个方向去找暑期科研，可以有2-3个感兴趣的相关的方向，孤注一掷容易翻车。最好做到能用1-2句英文句子表达清楚自己的研究兴趣。以往研究经历的简要描述：以往的研究经历能够展示出你大概的研究方向、已经具备的研究技能和对研究的一腔热血。每段研究经历当中需要包含的内容有：研究的主题+简单描述研究方法/研究技术+有趣的发现+自己的收获与思考。一些lab在招RA的时候会特别说明希望RA具备的技能和经验，例如发展心理学的lab很看重你和儿童相处的经验，还有心理学lab普遍看中的编程能力、数据分析能力，这些要求大概率都可以在你想找的lab的页面上找到。以往研究经历的描述最好是可以灵活变通套到模板里面的，尽量贴合目标lab的研究主题。一般来说，一个lab的网站上会有欢迎界面、研究介绍【Research】、研究团队【People】、发表情况【Publications】、新闻报道、参与研究【Participation】、加入团队【Join 1. 一份完善的英文简历（CV）。
   感兴趣的几个研究方向自己以往研究经历的简要描述，灵活易改动（使用方式类似于托福口语中准备的万能例子）。这个是非常重要的一步，我暑研的导师曾经告诉我每天ta都会收到无数的暑研申请，花在每封邮件上的时间只有几秒钟，这就意味着你需要把最重要的信息放在邮件里，保证邮件简洁明了，进一步意味着其他一些不怎么相关的信息需要放在你的简历中，如果教授感兴趣再去看你的简历了解更多关于你的事情。简历中必须包含的信息有：姓名、联系方式、目前就学情况、裱花（GPA+语言成绩+GRE）、参与过的研究项目简述、具备的研究技能，加分项有：获奖信息、奖学金信息等等。此外还要注意简历一定要大方整洁，什么花花绿绿的贴纸边框就别整上去了，如何准备一份得体的英文简历也是一门学问。简历一般作为邮件的附件附在后面，高端玩家会设计一个自己的个人网站充当简历的作用，取决于个人水平了。

2. 一个海投邮件模板。
   海投发邮件的量是很大的，看下来一般自助申请暑研的同学们发的邮件量都在100+，这么大的发件量注定了你需要一个开头结尾段落模板，不然每个都重新写真的要累死。给大家提供一个万能模板：第一段：我是谁，我来自哪儿，我发邮件来申请暑期科研机会第二段：我如何认识的贵lab，我对贵lab的什么研究感兴趣，匹配的研究经历，我的思考第三段：我来暑研的工作内容，我的技能如何对贵lab作出贡献第四段：我不要钱志愿来搬砖，我计划什么时候来第五段：谢谢谢谢，期待回信怎么灵活运用模板就要看个人的造化了。

3. 一个记录自己发邮件情况的excel表格。
   因为海投量很大，海投时长从12月开始到4月持续好几个月，这时候可千万不要相信自己的记忆。最好用一个表格记一下自己心仪的实验室，给投过的实验室做标记，以免发漏，之后第二遍追加投的时候也可以做标记，下面是我暑期科研的时候用来记录申请情况的一个excel表格，就包含了各种需要的信息，用颜色表明自己投过与否以及投了几遍。这个表格建议大家根据自己的情况设计。<img src="https://pic1.zhimg.com/50/v2-c533ee20fc23f96763a82c850270bd34_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1336" data-rawheight="448" data-default-watermark-src="https://pic1.zhimg.com/50/v2-6e9368166aad20fababd3ca521b1096f_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1336" data-original="https://pica.zhimg.com/v2-c533ee20fc23f96763a82c850270bd34_r.jpg?source=1940ef5c"/>


# 官方项目

此类暑期科研的申请，总结起来有以下特点：1）选项灵活，不受院校项目的限定，从常青藤到普通州立大学都可以列为申请目标；2）时间灵活，什么时候都可以开始，一般最晚在3-4月份仍有机会；3）流程简单，不需要太多申请材料和步骤，一般只需要准备语言成绩，套瓷信和个人简历，在和导师取得沟通和面试后即可确认参加。

此类申请的主要难点在于能否寻找到合适的教授，以及匹配合适的科研项目，需要投入大量的时间，但是结果回报不确定。
